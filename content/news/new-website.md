I'm happy to announce that my personal website has been completely redesigned and is now live.

The new site is built as a custom static site generator using Python. It supports three languages — English, French, and Bengali — and the content can be switched seamlessly between them from any page.

A few things that went into it : the layout is cleaner and more responsive, pages load faster thanks to optimised assets, and the code that generates the site is something I wrote with Claude Sonnet 4.6, rather than depending on a framework I was fighting with.

If you notice anything broken or have suggestions, feel free to get in touch.
